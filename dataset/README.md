# Datasets

Os datasets grandes **não são versionados no Git** (ver `.gitignore`). Baixe-os
manualmente e extraia nesta pasta antes de rodar os notebooks que os utilizam.

## Amazon Reviews (polarity)

Usado nas aulas de NLP (classificação de sentimento). Baixe de:

**https://www.kaggle.com/datasets/kritanjalijain/amazon-reviews**

Após extrair, a estrutura esperada é:

```
dataset/amazon_review_polarity_csv/train.csv   (~1.5 GB)
dataset/amazon_review_polarity_csv/test.csv    (~168 MB)
```
